# 快速开始指南

## 第一步：获取 Bot Token

1. 在 Telegram 中搜索 `@BotFather`
2. 发送 `/newbot` 创建新机器人，或发送 `/mybots` 查看已有机器人
3. 选择你的机器人，点击 `API Token`
4. 复制 Token（格式类似：`1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`）

## 第二步：获取你的用户ID

1. 在 Telegram 中搜索 `@userinfobot`
2. 发送任意消息
3. 机器人会回复你的用户ID（纯数字，例如：`123456789`）

## 第三步：配置机器人

### 方法1：创建配置文件（推荐）

1. 复制 `user_config.example.py` 文件，重命名为 `user_config.py`
2. 编辑 `user_config.py`，填入你的信息：

```python
BOT_TOKEN = '你的机器人Token'
ADMIN_USER_IDS = '你的用户ID'
```

**示例：**
```python
BOT_TOKEN = '1234567890:ABCdefGHIjklMNOpqrsTUVwxyz'
ADMIN_USER_IDS = '123456789'
```

如果有多个管理员，用逗号分隔：
```python
ADMIN_USER_IDS = '123456789,987654321'
```

### 方法2：设置环境变量

#### Windows PowerShell
```powershell
$env:BOT_TOKEN="你的机器人Token"
$env:ADMIN_USER_IDS="你的用户ID"
```

#### Windows CMD
```cmd
set BOT_TOKEN=你的机器人Token
set ADMIN_USER_IDS=你的用户ID
```

## 第四步：运行机器人

```bash
python main.py
```

如果配置正确，你会看到：
```
🤖 机器人启动中...
📋 管理员数量: 1
📦 检查数据库...
✅ 数据库已就绪
✅ 机器人已启动，等待消息...
```

## 常见问题

### Q: 提示 "BOT_TOKEN 未设置"
**A:** 请确保：
1. 已创建 `user_config.py` 文件
2. 文件中的 Token 格式正确（字符串，用引号包围）
3. 或者已设置环境变量

### Q: 提示 "ADMIN_USER_IDS 未设置"
**A:** 请确保：
1. `user_config.py` 中有 `ADMIN_USER_IDS` 配置
2. 用户ID是纯数字，多个ID用逗号分隔
3. 或者已设置环境变量

### Q: Token 无效错误
**A:** 
1. 检查 Token 是否完整（没有遗漏字符）
2. 在 @BotFather 中重新获取 Token
3. 确保 Token 没有被撤销
